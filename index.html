Pandas中的数据类型

  1. Series: 一维数组，但是每个数据都有自己的标签。
  2. Dataframe: 二维表格，每个轴都有标签。

csv file

csv (Comma-Separated Values) 指逗号分隔值，是一种通用的，相对简单的文件格式。一般的测定仪器都可以选择保存成这种格式的文件。

Pandas 中有直接可以读取 csv 文件的函数 “read_csv”。可以通过它把 csv 文件转化成 dataframe.

使用方法

有很多选项可以调整，这里面只介绍常用的，具体可以参考这里
read_csv说明。

  1. filepath_or_buffer: 需要读取的文件
  2. sep: 分隔符，默认的就是’,’
  3. header: 选择用来命名行标题的行数，并选择起始数据，默认是 0 。
  4. index_col：选择索引列，默认是无。
